# Getting Started

Welcome to `attack-on-domain`! This section will help you get up and running quickly.

## Overview

`attack-on-domain` is a Python library that provides Domain-Driven Design building blocks. It's built on top of Pydantic v2 and provides:

- **Domain Layer** — Entities, Value Objects, Services, Events
- **Application Layer** — Use Cases, Ports, Contracts
- **Infrastructure Layer** — Sessions, Projections, Containers
- **Testing Utilities** — Spies, Fakers, Event Assertions
